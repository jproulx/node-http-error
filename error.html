HTTPError

The HTTPError class extends the built-in Error object, and has the following custom members:

code The numeric HTTP status code

status The HTTP status string

"use strict"; var http = require('http'); var assert = require('assert'); var customError = require('custom-error-generator'); var reverse = {}; var names = {};

Public function createHTTPError

Create a custom error class based on an HTTP status code

Parameters:

  • code can be a Number or a String.
    (The HTTP status code to derive the error message from)

Returns a HTTPError

var HTTPErrors = require('errors');
throw new HTTPErrors.NotFoundError('/missing');
